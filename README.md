# Smart Sprinkler Irrigation System

## Overview
The Smart Sprinkler Irrigation System automates watering based on soil moisture. It helps conserve water and ensures plants receive the right amount of irrigation.

## Hardware Components
- Arduino Uno
- Soil Moisture Sensor
- Relay Module
- Water Pump
- Jumper Wires
- Breadboard
- Power Supply

## Working
1. The soil moisture sensor measures soil moisture.
2. Arduino reads the sensor value.
3. If the soil is dry, the relay turns ON.
4. The water pump starts watering the plants.
5. When sufficient moisture is detected, the pump turns OFF.
